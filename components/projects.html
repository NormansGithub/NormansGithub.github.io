MadWeather automates checking the weather as smoothly as possible. Uses the user's location and weekly schedule, it determines it retrieves the weather and notifies the user accordingly to their preferences.

This project was split between me and my friend. I implemented the events occuring after the user's alarm sets off. This included checking the weather online and notifying the user accordingly. I also implemented the user interface and added functionality to the buttons on the screen.

oChemDraw is a application simplifying how users draw molecules. Instead of drawing out the atoms individually, oChemDraw uses each molecule's unique IUPAC name to identify the features and functional groups present in the molecule.

The application was built from scratch using only Java libraries. The user input is decomposed into fragments of smaller molecules until they become basic functional groups. They are then aligned together to build the original molecule the user had in mind.
